# Guidelines checklist creation: template prompt

There are lots of guidelines and panel discussions on best practice for covering a range of issues, from suicide or protest to immigration or domestic abuse.

The BCU Journalism Style Guide [special considerations section](https://docs.google.com/document/d/1Bp0VGxRuD0TD4dT2W9c_1wkfjyebxkinpAWmC0E71Do/edit?tab=t.0#heading=h.o34hmwpdr8c3) contains links to lots of guidelines that can be used for this.

AI's translation abilities are particularly well suited to translating these into practical checklists.

This prompt can be used with documents but it's especially useful in NotebookLM for translating video discussions. To do this, select *Add sources* and paste the video URL. 

One video to test it with is [this one hour Ethical Journalism Network panel on Responsible Reporting on Domestic Violence](https://www.youtube.com/watch?v=cdwSFyTyDng&t=1s): `https://www.youtube.com/watch?v=cdwSFyTyDng&t=1s`

## Prompt template

```
Create a checklist that journalists can use for any stories they are producing relating to [TOPIC].
Make it suitable for a journalist who has written a draft and needs to check for specific mistakes or omissions.
Limit it to 10 items.
For each item indicate the source(s) and the page number/timestamp so I can explore guidance in context.
Make it succinct and practical.
```
