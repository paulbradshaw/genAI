# Reforms to securitisation requirements

This folder contains the consultation [CP2/26 – Reforms to securitisation requirements](https://www.bankofengland.co.uk/prudential-regulation/publication/2026/february/reforms-to-securitisation-requirements-consultation-paper) (printed as a PDF) and the five PDF appendices. 

Responses to the consultation can be added by searching for [`response to CP2/26 - Reforms to securitisation requirements filetype:pdf`](https://duckduckgo.com/?q=response+to+CP2%2F26+-+Reforms+to+securitisation+requirements+filetype%3Apdf&ia=web)
